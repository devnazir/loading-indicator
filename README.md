# loading-indicator
HOC Loading Indicator for my personal react project

### install
```
npm i @devnazir/top-indicator
```

### Example Usage
App.js
```
import WithLoadingIndicator from "@devnazir/top-indicator"

function App() {
  ...
}

export default WithLoadingIndicator(App)
```

index.js
```
<App colorIndicator="blue"/>
```

colorIndicator
1. blue
2. red
3. gold
4. chocolate
5. green
6. yellow
